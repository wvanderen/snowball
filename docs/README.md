## Current north star

Snowball is a local-first, mobile-first activity tracker disguised as an incremental lattice game. The daily-life loop should stay frictionless while the game layer turns real focus time and consistency into momentum, Energy, per-sphere XP, Core Power, glyph rewards, and a growing personal sigil.
